       - `fun fetchPrecatorio(numero: String): PrecatorioScraped`:
         Wrap with `RateLimiter.decorateCheckedSupplier(cacRateLimiter) { Retry.decorateCheckedSupplier(cacRetry) { doFetchPrecatorio(numero) }.apply() }.apply()`

       - `private fun doFetchPrecatorio(numero: String): PrecatorioScraped`:
         Step 1 - GET form page: `session.newRequest(properties.precatorioPortal.baseUrl).get()`
         Step 2 - Extract ViewState fields:
           - `formPage.select("input[name=__VIEWSTATE]").firstOrNull()?.attr("value") ?: ""`
           - `formPage.select("input[name=__VIEWSTATEGENERATOR]").firstOrNull()?.attr("value") ?: ""`
           - `formPage.select("input[name=__EVENTVALIDATION]").firstOrNull()?.attr("value") ?: ""`
         Step 3 - POST with ViewState + precatorio number:
           - `session.newRequest(properties.precatorioPortal.baseUrl).data("__VIEWSTATE", viewState).data("__VIEWSTATEGENERATOR", viewStateGen).data("__EVENTVALIDATION", eventValidation).data("numeroPrecatorio", numero).post()`
         Step 4 - Blank-form detection: if result page has empty result container (e.g., no `.resultadoPesquisa` elements):
           - If sessionRenewCount > 0, throw `ScrapingException("CAC blank form after session renewal for $numero â€” not retryable")`
           - Else: reset session = `Jsoup.newSession().timeout(...).userAgent(...)`, increment sessionRenewCount, throw `ScrapingException("CAC session expired for $numero â€” renewed, retrying")`
         Step 5 - Parse result page: extract entidadeDevedora, valorOriginal, valorAtualizado, natureza, statusPagamento, posicaoCronologica, dataExpedicao using CSS selectors. Use same graceful degradation pattern as EsajScraper (null + missingFields, per D-04/D-05/D-06)
         Step 6 - Return `PrecatorioScraped` with all fields and rawHtml
